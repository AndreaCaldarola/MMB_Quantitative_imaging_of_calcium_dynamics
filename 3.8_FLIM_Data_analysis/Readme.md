## Preparations

Unless you have carried out the 'protocol' that belongs to section 3.2, you will need to create a new environment in the terminal and install depencies:

#### create a new environment

* `conda create -n Sensor_Calibration`
* `conda activate Sensor_Calibration`

#### Install necessary dependencies


* `conda install jupyter`
* `conda install numba`
* `conda install pandas`
* `conda install scipy`


#### Run Jupyter

* `jupyter lab` or `jupyter notebook` and start "Stellaris_LifetimePlotter_Simple.ipynb"



`flimtools.py` and `linleastsquares.py` are required to run the notebook and are included in this folder. These scripts were originally created by Jay Unruh: [https://github.com/jayunruh/FLIM_tools](https://github.com/jayunruh/FLIM_tools)
